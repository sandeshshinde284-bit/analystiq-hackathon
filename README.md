## 🎯 The Problem
**VCs review 1,000+ deals annually, spending 23+ hours per analysis.** Traditional due diligence drowns investors in unstructured data—pitch decks, founder calls, emails—leading to missed opportunities and slow decisions.

## 💡 Our Solution
**AI-powered analyst** that reads, benchmarks, and flags risks in startup documents, delivering **investor-ready insights in under 3 minutes**.

## 🏆 Key Features

![Deal Memo Dashboard](images/dashboard-demo.png)

### 🤖 **AI-Powered Analysis**
- Multi-source ingestion (PDFs, transcripts, emails)
- Automated risk assessment & red flag detection
- Real-time benchmarking against 10,000+ startups

### 🔍 **Source of Truth Engine** *(Our Killer Feature)*
![Source Verification](images/source-modal.png)
- Every insight 100% traceable to original document
- Click any data point → See exact source location
- Ultimate transparency & trust building

### ⚙️ **Customizable Intelligence**
- Adjust analysis weights (Team, Market, Product, Traction)
- Real-time recommendation updates
- Tailored to your investment thesis
